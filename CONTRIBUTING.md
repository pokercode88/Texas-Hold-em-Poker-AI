# Contributing to Texas Hold'em AI System

感谢您考虑为这个项目做出贡献！我们欢迎任何形式的贡献，包括报告问题、提交代码、改进文档等。

## 行为准则

参与本项目即表示您同意遵守我们的[行为准则](CODE_OF_CONDUCT.md)。

## 如何贡献

### 报告 Bug

如果您发现了 Bug，请通过 GitHub Issues 报告，并包含以下信息：

- 清晰的问题描述
- 复现步骤
- 预期行为和实际行为
- 环境信息（OS、Python/C++版本等）
- 相关日志或截图

### 提交功能请求

欢迎提交新功能建议。请描述：
- 该功能解决什么问题
- 可能的实现方案
- 是否愿意参与实现

1. **Fork 仓库**

   # 点击 GitHub 上的 Fork 按钮
   # 然后克隆您的 Fork
